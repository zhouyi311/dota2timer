# Dota2 Timer

A real-time, browser-based event timer for Dota 2, designed to provide timely audio alerts for crucial in-game events. This tool helps players improve game sense and efficiency by offloading the mental load of tracking timers.

[**Live Application**](https://dota2timer.ez2run.com/)

## ✨ Features

- **Real-time Event Tracking**: Monitors periodic events like Jungle Stacks, Power Runes, Lotus/Bounty, and Wisdom Runes.
- **Customizable Audio Alerts**: Provides both simple sound cues (dings) and voiceovers for each event.
- **Configurable Lead Times**: Users can set how many seconds in advance they want to be notified for each event.
- **Multi-language Support**: Fully internationalized (i18n) with support for English, Spanish, Russian, and Chinese. Audio voiceovers also change with the selected language.
- **Role-based Presets**: Quickly configure event tracking based on standard Dota 2 positions (pos1, pos2, etc.).
- **Persistent Settings**: All user preferences are saved locally in the browser's `localStorage`.
- **Two-Phase Timer**: Supports both a "Strategy Phase" (pre-game countdown) and the main "Game Phase".
- **Responsive Design**: Clean and functional layout on both desktop and mobile devices.

---

## 🚀 Getting Started

1.  Open the application in your browser.
2.  Use the "Presets" to quickly select alerts for your role, or manually toggle each event.
3.  Adjust lead times and voiceover preferences for each event.
4.  Click "Start" to begin the timer.
